# Personal Portfolio Page

A portfolio website built as part of Day 2 as an intern at SprintX for Practice

## Files

- `index.html` — page structure and content
- `style.css` — styling for the page
- `README.md` — this file

## Sections included

- Header with navigation
- Intro with name, tagline, and profile placeholder
- About section
- Skills section
- Experience & education table
- Projects section (4 project cards)
- Contact section with a working contact form (name, email, message, submit)
- Footer

## HTML concepts used

- Semantic tags: `header`, `main`, `section`, `footer`
- Headings, paragraphs, and lists
- A table for experience/education
- A form with text, email, and textarea inputs and a submit button
- Links (nav, social, contact)

## CSS concepts used

- External stylesheet linked to `index.html`
- Custom color palette using CSS variables
- Google Fonts (Poppins + Inter)
- Margin/padding based spacing
- Borders and rounded corners
- Styled buttons with hover states
- Flexbox (nav, skills, tags) and CSS Grid (about, projects, contact)
- Media queries for tablet (768px) and mobile (480px) breakpoints
